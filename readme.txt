Official Affilicious Theme for Wordpress with product management, price comparisons, details and more.

== Changelog ==
= 0.7.2 =
* Added page templates like full-width, page-builder, sidebar-left and sidebar-right.
* Simplified the source directory structure.

= 0.7.1 =
* Fixed a bug related to a missing class

= 0.7 =
* Made the theme compatible with the beta 0.8
* Bug fixes

= 0.6.3 =
* Made the product details table more mobile friendly.
* Fixed the broken sidebar setup on new installations.
* Removed some old Affilicious Plugin dependencies.

= 0.6.2 =
* Added options to hide the "test review button" and "buy button".
* Fixed some update issues.

= 0.5.1 =
* Fixed a critical bug after the theme activation on missing or outdated affilicious plugin
* Changed the requirement of the affilicious plugin to 0.6

= 0.5 =
* Added support for product variants and attributes

= 0.4.3 =
* Added the possibility to define click actions on product images
* Added missing no-follow and target-blank properties to affiliate links
* Removed the weird "lala" label from empty prices

= 0.4.2 =
* Added support for change logs
* Set affiliate links to no-follow and target-blank
* Added affiliate links to the product image gallery
* Changed the product sidebar to the default one
* Many optimizations and cleanups
* Added Gulp for easier and faster development
